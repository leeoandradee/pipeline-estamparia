Silk Screen

A técnica mais utilizada para personalização de camisetas

Indicação: Camisetas, Polos, Moletons

Tecidos aceitos: O Silk Screen é a técnica mais versátil disponível, aceita praticamente todos os tipos de tecido.

Limite de Cores: Não há limite de cores, porém quanto mais cores, mais elevado será o custo, já que cada cor demanda a produção de uma tela.

Limitação de Área: 28x40cm

Pedido mínimo: 12 peças

Durabilidade: Seguindo as recomendações de lavagem, as estampas tem durabilidade indeterminada.

Sublimação

Técnica utilizada para estamparia de tecidos brancos sintéticos e canecas

Indicação: Qualquer tecido branco 100% poliéster

Tecidos aceitos: A sublimação só é aceita por tecidos 100% poliéster

Limite de Cores: Não há limite de cores

Limitação de Área: Tamanho A3 42×29,7cm

Pedido mínimo: Uma peças

Durabilidade: As estampas tem durabilidade indeterminada.

DTF

Uma das mais modernas técnicas de estamparia para camisetas personalizadas DTF vem de direct-to-film, ou seja, você imprime sua arte em um filme especial com tintas têxteis para impressoras.

Indicação: Camisetas, Polos, Moletons, Dryfit, Malha fria, etc... Recomendamos para baixa tiragem

Tecidos aceitos: O DTF é uma técnica versátil recomendado para baixa tiragem, aceita praticamente todos os tipos de tecido.

Limite de Cores: Não há limite de cores.

Limitação de Área: 28x40cm

Pedido mínimo: 1 peças

Durabilidade: Seguindo as recomendações de lavagem, as estampas tem durabilidade indeterminada.
